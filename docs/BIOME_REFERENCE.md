## **Technical Usage in Generator**

### **Layer System (Pre-1.18)**
1. **Category Assignment**: The game first picks `Snowy`, `Cold`, `Medium`, or `Warm`.
2. **Biome Choice**:
   - `Warm` -> Desert (2), Savanna (35), Plains (1).
   - `Medium` -> Plains (1), Forest (4), Dark Forest (29), Swamp (6), Birch (27).
   - `Cold` -> Forest (4), Mountains (3), Taiga (5), Plains (1).
   - `Snowy` -> Snowy Tundra (12), Snowy Taiga (30).
3. **Mutation**: A 1/128 RNG check runs to transform `Biome ID` to `Biome ID + 128`.

### **Multi-Noise (1.18+)**
There are no "Categories". Each biome has a 6D point.
*   **Windswept Savanna** (Shattered Savanna) is no longer a mutation of Savanna. It is its own biome defined by `High Erosion` and `High Weirdness`.
